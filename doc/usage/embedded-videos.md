# Embedded Videos

## Post a YouTube Video

1. Copy a standard YouTube video link (e.g., `https://www.youtube.com/watch?v=...` or `https://youtu.be/...`).
2. Paste the link into the message box on its own line, like this:

```markdown
[youtube video](https://www.youtube.com/watch?v=...)
```

3. Post the message.
4. The application will automatically convert the link into a responsive, embedded video player.

## Supported YouTube URL Formats

- Standard format: `https://www.youtube.com/watch?v=VIDEO_ID`
- Short format: `https://youtu.be/VIDEO_ID`
- Embed format: `https://www.youtube.com/embed/VIDEO_ID`

## Tips

- Make sure the YouTube link is on its own line in your message
- The video player is fully responsive and works on all devices
- You can add text before or after the video link in the same message
- The embedded player supports all standard YouTube features (playback controls, fullscreen, etc.)
